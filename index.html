fleet-xrdp

Custom xrdp 0.10.6 + matching xorgxrdp with RemoteFX and H.264, built per-distro and served as a GPG-signed apt repository. Suites: noble jammy kali-rolling (amd64).

Install

curl -fsSL https://norandom.github.io/xrdp-fleet/install-client.sh | bash

Auto-detects the Ubuntu base (noble / jammy; Linux Mint supported), adds this repo with GPG verification and an apt pin, then installs xrdp + xorgxrdp.

Contents

  • install-client.sh
  • pubkey.gpg — repo signing key
  • dists/<suite>/ + pool/<suite>/ — the archive

Source & docs: github.com/norandom/xrdp-fleet
